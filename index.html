Base VIM
Pour entrer dans Vim (à l’invite $) tapez vim [Fichier] entrée
Pour quitter Vim sans enregistrer tapez Échap :q ! entrée
Pour quitter Vim en enregistrant tapez Échap :x
Pour insérer du texte au niveau du curseur en mode normal tapez i Vous êtes alors dans un mode insertion
Pour quitter tapez Échap
Effacement
et annulation
Pour effacer un caractère sous le curseur en mode normal tapez x
Pour effacer du curseur jusqu’à la fin d’un mot tapez dw
Pour annuler une commande partiellement tapée dont vous ne voudriez plus
tapez Échap
Pour effacer du curseur jusqu’à la fin d’une ligne tapez d$
Pour effacer toute une ligne tapez dd
Le format d’une commande en mode normal est : [nombre] commande objet où
nombre : c’est combien de fois est répétée la commande.
commande : c’est ce qu’il faut faire, par exemple d pour effacer
objet c’est ce sur quoi la commande devrait agir, par exemple l (letter),
Pour annuler des actions précédentes tapez u comme undo
Pour annuler tous les changements sur une ligne tapez U majuscule.
